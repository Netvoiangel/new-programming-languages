## Алгоритмическая задача (Julia, Ruby, Groovy)

Одна и та же задача во всех трёх: **решето Эратосфена** — вывести простые числа до \(n\).

### Julia
- Файл: `julia/main.jl`
- Запуск: `julia main.jl 100`

### Ruby
- Файл: `ruby/main.rb`
- Запуск: `ruby main.rb 100`

### Groovy
- Файл: `groovy/Main.groovy`
- Запуск: `groovy Main.groovy 100`
