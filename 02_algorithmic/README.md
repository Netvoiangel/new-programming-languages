## Алгоритмическая задача (Julia, Ruby, Groovy)

Одна и та же задача во всех трёх: **задача о N ферзях (N-Queens)** — рекурсивный бэктрекинг.

Что делает программа:
- считает **количество решений** для доски \(N \times N\),
- печатает **одну** найденную расстановку (если есть),
- выводит простую статистику (время выполнения).

### Julia
- Файл: `julia/main.jl`
- Запуск: `julia main.jl 12`

### Ruby
- Файл: `ruby/main.rb`
- Запуск: `ruby main.rb 12`

### Groovy
- Файл: `groovy/Main.groovy`
- Запуск: `groovy Main.groovy 12`
